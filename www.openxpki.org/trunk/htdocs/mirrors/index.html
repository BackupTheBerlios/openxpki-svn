OpenXPKI Mirror

OpenXPKI uses a set of www mirrors to distribute the projects website.
The current development is done on the berlios svn. Anonymous access to the repository is possible if you want to look at the current development code. Currently there are no snapshots or tarballs available - we will announce their date and kind of distribution as soon as we decide to publish them here on the website an through our Mailinglists (see resources).

Mirrors of the OpenCA Project

The former www mirror of the openca.info website is superseded by this website. There is a ftp mirror system for releases of OpenCA - you can find a mirror close to you here.

Establishing a Mirror

If you want to establish a www mirror for the OpenXPKI website, see the www mirror section for details.

If you want to establish ftp mirrot for OpenCA release, please refer to the openca.org website.
