WWW Mirrors

The website you are currently reading is maintained via the berlios svn repository and hosted on several independant www servers to increase availability.
If you want to mirror these pages you simply have to put a small script onto your webserver, that pulls a copy from the repository.
You can get the script from the svn archives to, see Ressources section for details.
If you established a mirror please send a mail to us via openxpki-users@berlios.de so we can add you to this list and assign a dns entry
