WWW Mirrors

You can mirror openca.info by CVS. We added all of our web stuff to OpenCA's CVS on SourceForge.net. There exists a module called www.openca.org. This module includes an example script update_www.openca.org which demonstrates how to mirror the OpenCA pages. If you mirror OpenCA then please contact us via openca-users@lists.sf.net or openca-devel@lists.sf.net. We will add you to our mirror list and you can get an OpenCA DNS entry too.
